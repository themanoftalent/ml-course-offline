Machine learning data basics

Models learn from data. Each row describes an observation, each column is a feature, and labels mark the target you want to predict. Clean, well-structured data improves results more than adding complexity to the model.

Common data types

  • Numerical. Measured or counted values. Examples: temperature, price, number of items. Continuous values fill a range. Discrete values are counts.
  • Categorical. Group labels. Examples: city, product category, blood type. Nominal has no order. Ordinal has a natural order such as small, medium, large.
  • Boolean. True or false flags. Example: is fraud, yes or no.
  • Text. Words and documents that you convert to numerical features. Examples: reviews, emails, transcripts.
  • Image, audio, video. Rich media often used with deep learning. Examples: X-ray images, voice commands, traffic videos.

Convert everything to model-ready numerical features. Keep data consistent, handle missing values, and check distributions before training. Good data beats a fancy model with poor inputs.
